Homebrew Test Equipment Calibration

Welcome to the bootstrap calibration lab. This project provides detailed, homebrew calibration procedures for common test equipment without requiring any pre-calibrated reference equipment.

The Bootstrap Calibration Philosophy

"How do you calibrate test equipment when you have no calibrated equipment?"

We use fundamental physical constants and freely available reference signals:

  • WWV/WWVH - NIST time signals (2.5, 5, 10, 15, 20 MHz) - frequency accuracy better than 1×10⁻¹²
  • GPS 1PPS - GPS satellites provide 1 pulse-per-second timing (±40 ns accuracy)
  • AC Mains - Power line frequency (60 Hz in North America, ±0.02 Hz)
  • Precision resistors - 0.1% tolerance resistors as voltage/current references
  • Crystal oscillators - Can be disciplined against WWV or GPS
  • Known physics - Speed of light, resonant frequency calculations

These methods achieve 0.1% - 1% accuracy, which is excellent for amateur radio work and often better than commercial "calibrated" instruments!
